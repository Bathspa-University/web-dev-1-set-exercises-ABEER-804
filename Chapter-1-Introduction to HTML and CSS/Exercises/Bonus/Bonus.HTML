Quick Reference Guide for HTML and CSS resource.

In this guide you will have a clear overview on which HTML tags and CSS properties are used for purposes and when to use them.

Lets starts with HTML tags use in the HTML document:

Head element:

The Head element is used to in the start of the HTML file which contains all the the meta information of the document.

Title element:

The Title element gives heading on the top left of the windows tab. Specifies the title of the HTML page.

Body element:

The Body tag is used to define the document of the body and the contents in it. Hyperlinks, images, tables, headings, paragraphs, list.

p tag:

The p tag is use for paragraphs.

img tag:

The img tag is used for images with src attribute which give the location of the file in the folder and does not have a closing tag.

Comment tag:

The comment tag is used for defining the content and reminders in the HTML files(They are not displayed in HTML document).

q tag:

With q tag the user can insert in short quotations in the body.

table element:

The table element consists of rows and columns, which contains table data

a tag:

An a tag is used to insert links in the href attribute to open the link to another site.

ordered and unordered list element:

An ordered and unordered list element is used to create list in the HTML document, using ul and ol tags which insert under li tag for list items.

Properties used in CSS to style the HTML document:

The background-color property:

This property gives HTML document a color in the background to make it attractive.

font-family property:

This property gives text a different style in the document. 

color property:

The color property is used to give color to the text in the HTML document.

text decoration property:

This property is used to add decoration line to the text.

padding property:

generates space around element's content.

text-align property:

This property is used to align text in the HTML document.

CSS outline property:

the outline property is line drawn around the element to make the it stand out. 

text-transform property:

This property is used to specify the text in uppercase and lowercase or capital.
